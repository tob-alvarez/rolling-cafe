
# Rolling Cafe

This is a static coffe shop page build with ReactJS + Vite, Bootstrap, dbjson, and love. 


## Installation

Install my-project with npm

```bash
    git clone https://github.com/tob-alvarez/rolling-cafe.git
    cd rolling-cafe
    npm install
```
Run json-server

```bash
    json-server --watch ./db.json
```
Run the app

```bash
    npm run dev
```
